## 목록 (list)

### 순서가 없는 리스트 : `-`(hypen), `*`(asterisk)

목록 활용시 단계를 `tab`과 `shift + tab`으로 조절한다.

### 순서가 있는 리스트 : 1.

## 코드 블록

### Fenced Code Block

- `(backtick) 기호 3개를 활용(''')
- 특정 언어를 명시하면 Syntax highlighting 기능이 적용된다.

```
print('hello')
```

### Inline Code block

- `(backtick) 기호 1개 활용

`print` 는 파이썬에서 출력하는 함수이다.
